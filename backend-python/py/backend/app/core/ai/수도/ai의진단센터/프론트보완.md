### 프론트엔드 UI/UX 보완점 (AI 진단 고도화용)

#### 1. 진단 결과 페이지 (Diagnosis.jsx)
*   **[섹션 분리]**: 단순 `summary` 텍스트 출력에서 벗어나, AI가 제공하는 **[위험 총평 / 피해 예측 / 긴급 조치]**를 각각 별도의 카드나 아코디언 컴포넌트로 시각화.
*   **[로딩 화면 강화]**: 스프링과 파이썬을 거쳐 분석하는 동안 사용자에게 **"AI가 설문 답변을 기반으로 정밀 분석 중입니다..."**와 같은 진행 상태를 더 역동적인 애니메이션으로 노출.

#### 2. 마이페이지 진단 센터 (DiagnosisCenter.jsx)
*   **[데이터 매핑]**: 
    *   `history` 목록: 백엔드의 새 API와 연결하여 실제 **진단명, 날짜, 점수**가 테이블에 출력되도록 수정.
    *   상세 리포트: 선택된 항목의 고유 ID를 통해 DB의 `ai_comment`(총평)를 요약 탭에 출력.
*   **[체크리스트 동적화]**: 
    *   권장사항 탭에서 하드코딩된 예시 데이터를 제거하고, `AI_DIAG_RECOMMEND` 테이블에 저장된 **실제 AI 맞춤 처방전**을 불러와 체크박스 리스트로 렌더링.

#### 3. API 전송 데이터 (useDiagnosis.js / aiServiceApi.js)
*   **[검증]**: 현재 구현된 `question_text`와 `answer_text`가 누락 없이 백엔드로 전송되는지 재확인.
*   **[추가]**: 진단 시작 시 선택된 **`diagnosisName` (진단 종류)** 정보를 전송 객체에 포함하도록 보완.
